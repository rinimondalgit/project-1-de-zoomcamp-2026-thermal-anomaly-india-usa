# GitHub push checklist

1. Create a new empty GitHub repository.
2. Copy this project into your local repo folder.
3. Run:

```bash
git init
git branch -M main
git add .
git commit -m "Initial DE Zoomcamp 2026 final project"
git remote add origin https://github.com/<your-username>/de-zoomcamp-2026-thermal-anomaly-india-usa.git
git push -u origin main
```

4. Add screenshots to:
- `dashboard/`
- `images/`

5. Update README with:
- your BigQuery screenshots
- Airflow DAG screenshots
- Looker Studio screenshots
- final video link
